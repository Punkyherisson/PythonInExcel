## 🧪 4. Exercices pratiques recommandés

* Reproduire une **analyse Excel classique (somme, moyenne, tri)** avec `pandas`.
* Créer un **rapport Excel automatisé** à partir d’un CSV avec `openpyxl`.
* Automatiser des **graphiques dynamiques Excel** avec `xlsxwriter`.
* Contrôler **Excel en direct** avec `xlwings` (ex. : générer un tableau via Python dans un fichier déjà ouvert).

---

## 🔧 5. Packages Python essentiels à maîtriser

* `pandas` → manipulation de données tabulaires.
* `openpyxl` → lecture/écriture directe dans Excel.
* `xlsxwriter` → création de fichiers Excel avec formatage.
* `xlwings` → interaction directe avec Excel (idéal si tu l’as sur ton PC).
* `pywin32` (Windows uniquement) → contrôle d’Excel via COM.
